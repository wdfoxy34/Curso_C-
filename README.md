# Curso_C-

API em C# para comercialização de cursos online.
## Descrição
Este projeto é uma API desenvolvida em C# para gerenciar a comercialização de cursos online. A API permite a criação, leitura e exclusão de cursos, bem como a gestão de usuários e suas inscrições nos cursos, assim como o processamento de admin.istração dos cursos oferecidos. 
## Funcionalidades
- Gerenciamento de cursos (criação, leitura, exclusão)
- Gerenciamento de usuários (criação, leitura, exclusão)
- Inscrição de usuários em cursos

## Tecnologias Utilizadas
- C##
- ASP.NET Core
- Entity Framework Core
- SQL Server
